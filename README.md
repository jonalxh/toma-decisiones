# Toma de decisiones

Esta pequeña aplicación resuelve operaciones de toma de decisiones aplicando los siguientes criterios:

_Small app to solve decision-making operations applying the following criteria:_

- Wald (Maximin)
- Maximax
- Hurwitz
- Laplace
- Savage
